# Cochrane Statistical Data Packages

Save manually downloaded Cochrane "Download statistical data" files here.

Expected files include:

- `*.zip` RevMan/Cochrane data packages
- `*.rm5` older RevMan files
- `*.xml` Cochrane/RevMan XML exports, if available

Do not commit downloaded Cochrane packages unless you have confirmed the license permits redistribution. This directory is ignored by Git except for this README.
